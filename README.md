# Pure JavaScript PPTX Viewer 🎨

ブラウザ上でPowerPoint (.pptx) ファイルを表示する、Pure JavaScriptベースのビューアーです。サーバーサイド不要で、完全にクライアントサイドで動作します。

## ✨ 特徴

- **Pure JavaScript** - フレームワーク不要、バニラJSのみで動作
- **サーバー不要** - 完全にブラウザ上で処理（ファイルはサーバーに送信されません）
- **テーブル対応** - PowerPointの表を正しくレンダリング
- **テーマカラー対応** - スライドマスター・レイアウトのテーマカラーを反映
- **ズーム機能** - 拡大・縮小・リセット対応
- **レスポンシブ** - ウィンドウサイズに合わせて自動フィット
- **ダークテーマ** - 目に優しいダークモードUI

## 🚀 使い方

1. `index.html` をブラウザで開く
2. 「ファイルを選択」ボタンをクリック
3. `.pptx` ファイルを選択
4. スライドが表示されます

### 操作方法

| 操作 | 説明 |
|------|------|
| ◀ / ▶ | 前後のスライドに移動 |
| + / - | ズームイン / ズームアウト |
| % ボタン | ズームリセット |

## 🔧 技術詳細

### 依存ライブラリ

- **[JSZip](https://stuk.github.io/jszip/)** (v3.10.1) - PPTXファイル（ZIP形式）の解凍に使用（CDN経由で読み込み）

### 対応要素

- テキストボックス（フォントサイズ、太字、イタリック、色指定）
- 画像（PNG, JPEG, SVG, GIF, BMP, WebP）
- テーブル（セル結合、背景色、テキスト配置）
- 図形（背景色付き）
- グループ化されたオブジェクト
- スライドマスター / レイアウトの継承
- テーマカラー（明度調整含む）

### セキュリティ対策

- ファイルサイズ制限（200MB）
- ファイル拡張子バリデーション
- テキスト表示に `textContent` を使用（XSS防止）
- `"use strict"` モードの有効化
- ファイルは全てクライアントサイドで処理（外部送信なし）

## 📋 動作環境

- モダンブラウザ（Chrome, Firefox, Safari, Edge の最新版）
- JavaScript が有効であること
